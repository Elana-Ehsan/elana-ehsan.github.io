如何创建自己的blog--基于Github+Hexo

No tag
Publish Date:   2023-01-03

一、概述

可以参考以下文章，我就是参考这个文章，很多内容也来源于以下链接

GitHub+Hexo 搭建个人网站详细教程 - 知乎 (zhihu.com)

1、Hexo

Hexo是一种快速、简洁高效的博客框架

主要功能：搭建博客的前端页面框架，并附带部署功能，可以一键部署到网页上

官网：Hexo（配置完成之后再看会很明了，更细节化）

2、Github

全球最大的开源平台，可以放自己把自己的代码放上去，而且能给你配置好网页。

3、Node.js

我也不是太懂是干什么用的，以后学习再了解

4、Git

用来处理项目的，可以把自己的项目上传到github等平台上，也可以方便下载安装一些项目（也是还没学习，不太清楚）

Git教程 - 廖雪峰的官方网站 (liaoxuefeng.com)

Git - Downloading Package (git-scm.com)

二、配置部署

1、安装node.js

下载地址：Node.js (nodejs.org)

注意选择LTS版本，这个可以自动配置好PATH环境变量，安装一条龙。

安装完之后win+R，cmd

node -v

查看node版本；

npm-v

查看包管理器的版本。

还可以安装一个国内的包管理器cnpm

npm install -g cnpm --registry=https://registry.npm.taobao.org

查看cnpm版本

cnpm -v

2、安装Git

Git - Downloading Package (git-scm.com)

批处理，方便处理代码

3、创建博客文件夹，安装hexo

先创建一个文件夹用来放自己的blog

再右键打开

输入

npm install -g hexo-cli

或者

cnpm install -g hexo-cli

用来把hexo安装到blog文件夹中

注意：这里的命令都是作用再blog文件夹中

用hexo -v确认是否安装成功

安装完成后，初始化我们的博客，输入：

hexo init blog

可以用

hexo s

根据提示用网页打开预览（只是在本机上预览，并没有上传）

4、常用的hexo命令

注意：都是作用在blog文件夹中，可以右键用Git Bash Here打开

现在来介绍常用的Hexo 命令

npm install hexo -g #安装Hexo
npm update hexo -g #升级
hexo init #初始化博客

命令简写
hexo n “我的博客” == hexo new “我的博客” #新建文章
hexo g == hexo generate #生成
hexo s == hexo server #启动服务预览
hexo d == hexo deploy #部署

hexo server #Hexo会监视文件变动并自动更新，无须重启服务器
hexo server -s #静态模式
hexo server -p 5000 #更改端口
hexo server -i 192.168.1.1 #自定义 IP
hexo clean #清除缓存，若是网页正常情况下可以忽略这条命令

刚刚的三个命令依次是新建一篇博客文章、生成网页、在本地预览的操作。

5、部署网站——以Github为例

（1）配置SSH

之所以设置GitHub密钥，是通过非对称加密的公钥与私钥来完成加密，公钥放置在GitHub上，私钥放置在自己的电脑里。GitHub要求每次推送代码都是合法用户，所以每次推送都需要输入账号密码验证推送用户是否是合法用户，为了省去每次输入密码的步骤，采用了ssh，当你推送的时候，git就会匹配你的私钥跟GitHub上面的公钥是否是配对的，若是匹配就认为你是合法用户，则允许推送。这样可以保证每次的推送都是正确合法的。

第一步：在Git中获取ssh

右键Git Bash Here

git config --global user.name "你的GitHub用户名" git config --global user.email "你的GitHub注册邮箱"

设置配置信息

生成ssh密钥：

ssh-keygen -t rsa -C "你的GitHub注册邮箱（随便一个邮箱都可以，这里是方便记忆）"

然后直接三个回车即可，默认不需要设置密码
然后找到生成的.ssh的文件夹中的id_rsa.pub密钥，将内容全部复制

第二步：打开Github添加SSH密钥

右上角头像设置——SSH公钥

可以参考以下文章

生成/添加SSH公钥 - Gitee.com

用ssh git@github.com来判断是否成功建立连接（根据提示有可能要输入yes）

这样我们就把电脑和github建立连接了

2、文档编辑

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。Markdown语法简洁明了、容易掌握，而且功能比纯文本更强，因此写博客使用它，可以让用户更加专注的写文章，而不需要费尽心力的考虑样式，相对于html已经算是轻量级语言，像有道云笔记也支持Markdown写作。并且Markdown完全兼容html，也就是可以在文章里直接插入html代码。比如给博文添加音乐，就可以直接把音乐的外链html代码插入文章中。具体语法参看：Markdown 语法说明(简体中文版) 可以说十分钟就可以入门。可以用VScode查看。【摘自知乎】

而编辑文档的替代方案：Typora，我用的就是Typora，不用学什么md语法，直接编辑直接转，特别好用。
